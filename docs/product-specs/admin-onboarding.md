# Admin onboarding

## Goals

1. Configure `KWEAVER_BASE_URL` or use `kweaver-admin config set baseUrl <url>`.
2. Authenticate (`kweaver-admin auth login` or env token).
3. Verify access: `kweaver-admin auth status` and a read-only command (e.g. `user list`).
